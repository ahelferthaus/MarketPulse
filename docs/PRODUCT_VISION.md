# Westwood MarketPulse — Product Vision

## Mission
MarketPulse is Westwood's proprietary market sentiment and risk appetite platform — a visually compelling, rigorous, and transparent system that helps wealth clients, financial advisors, investment professionals, and market-watchers understand the emotional and positioning state of financial markets.

## Why MarketPulse?
Existing sentiment indices are either:
- **Opaque**: Black-box calculations with no transparency
- **Narrow**: Focus only on one dimension (e.g., just volatility, just surveys)
- **Static**: Updated infrequently with no historical context
- **Generic**: Indistinguishable from competitor products

MarketPulse differentiates through:
1. **Four-dimensional analysis**: Classic (market data), Narrative (text sentiment), Positioning (flows & positioning), Composite (blended)
2. **Transparent methodology**: Every component score documented and auditable
3. **Finance-media presentation**: MarketWatch/Barron's-quality visual storytelling
4. **Professional data readiness**: Built for Bloomberg, FMP, Morningstar integration
5. **Configurable architecture**: Swap data sources, adjust weights, add markets

## Target Audiences

### Public / MarketWatch-Style Readers
- Want a quick, visually compelling market psychology snapshot
- Need plain-English explanations, not jargon
- Appreciate "why it matters" context

### Wealth Clients
- Want to understand market conditions in meetings with advisors
- Need confidence indicators (is this data reliable?)
- Appreciate regime labels: "Markets are defensive today"

### Financial Advisors
- Use MarketPulse as a conversation starter with clients
- Need historical context: "What happened the last time we were in MP-1?"
- Want embeddable content for their own websites

### Internal Investment Professionals
- Need full component breakdown and raw data
- Want to understand what changed and why
- Require source quality badges and data confidence metrics
- Will connect Bloomberg/professional data sources

## Core Value Proposition
> **"Understand what markets are feeling, not just what they're doing."**

MarketPulse quantifies the psychological state of markets across multiple dimensions, making it actionable for both public audiences and professional users.

## Product Principles
1. **Simple at the surface, rigorous underneath**: Public users see a score and a sentence. Professionals see every component.
2. **Transparency builds trust**: Methodology is fully documented. Scores are auditable.
3. **Data confidence is non-negotiable**: Always show source quality, staleness, and substitutions.
4. **No false precision**: Scores are directional tools, not trading signals.
5. **Beautiful but not gimmicky**: Finance-media aesthetic, not game-like.

## Success Metrics
- **Engagement**: Time on page, return visits
- **Understanding**: Can a non-professional explain what the score means?
- **Trust**: Do professionals use the component data in their work?
- **Integration**: Is the embed widget used? Are exports consumed?

## Roadmap Summary
- **Now (MVP)**: Working local app with mock/public data, full scoring, React frontend
- **Q2**: Premium data integrations (FMP, paid news), embed widget live
- **Q3**: Bloomberg MCP connection, professional data mode, internal deployment
- **Q4**: Additional markets (sectors, international), prediction markets, mobile app
